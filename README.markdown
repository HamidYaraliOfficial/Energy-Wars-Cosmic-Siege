# 🚀 Energy Wars / Cosmic Siege

A futuristic 2D artillery strategy game built with Python and PyQt6. Command advanced factions, launch powerful missiles, adapt to environmental effects, and destroy enemy bases across dynamic battlefields.

---

# 🇬🇧 English

## Overview

**Energy Wars (Cosmic Siege)** is a science-fiction artillery game developed with **Python** and **PyQt6**. Players control a futuristic combat platform and engage enemy factions using a diverse arsenal of missiles with unique abilities.

### Features

- Modern PyQt6 graphical interface
- Dynamic physics simulation
- Wind and gravity system
- Multiple factions
- Multiple missile classes
- Special missile abilities
- Particle and explosion effects
- Mission progression system
- Upgrade and credit system
- Shop system
- Dark and Light themes
- Multi-language support (English, Persian, Chinese)
- Responsive interface

---

## Installation

### Requirements

- Python 3.10 or newer

### Install Required Libraries

```bash
pip install PyQt6
```

Or:

```bash
python -m pip install PyQt6
```

### Run the Game

```bash
python "Energy Wars.py"
```

---

## Gameplay

1. Select your faction.
2. Choose a missile.
3. Adjust firing power.
4. Aim using the mouse.
5. Launch missiles and destroy enemy bases.
6. Earn credits and unlock stronger weapons.
7. Progress through increasingly difficult levels.

---

## Missile Categories

### Light Missiles
Fast, agile, and low-cost.

### Medium Missiles
Balanced damage and speed.

### Heavy Missiles
Massive destruction and blast radius.

### Special Missiles
Unique abilities and advanced effects.

### Smart Missiles
Target-seeking and intelligent guidance systems.

# 🇨🇳 中文

## 项目简介

**Energy Wars（能量战争）** 是一款基于 **Python** 与 **PyQt6** 开发的科幻弹道射击游戏。玩家将操控未来战争平台，使用多种导弹摧毁敌方基地并完成任务。

### 功能特色

- 基于 PyQt6 的现代图形界面
- 动态物理模拟
- 风力与重力系统
- 多阵营支持
- 多种导弹类型
- 特殊导弹技能
- 爆炸与粒子特效
- 关卡成长系统
- 商店与升级系统
- 深色与浅色主题
- 多语言支持
- 响应式界面设计
